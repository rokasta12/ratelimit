| Package                   | Description                                  |
| ------------------------- | -------------------------------------------- |
| `@jf/ratelimit`           | Core library with algorithms and MemoryStore |
| `@jf/ratelimit-hono`      | Hono middleware                              |
| `@jf/ratelimit-express`   | Express middleware                           |
| `@jf/ratelimit-h3`        | H3/Nitro middleware                          |
| `@jf/ratelimit-nuxt`      | Nuxt module                                  |
| `@jf/ratelimit-unstorage` | unstorage adapter for distributed stores     |

## Quick Example

```ts
import { Hono } from "hono";
import { rateLimiter } from "@jf/ratelimit-hono";

const app = new Hono();

app.use(
  rateLimiter({
    limit: 100,
    windowMs: 60_000, // 1 minute
  }),
);

app.get("/", (c) => c.text("Hello!"));
```
